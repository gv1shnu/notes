Compression & Archiving

Difficulty: Intermediate

Compression reduces the amount of storage required by data, while archiving combines multiple files into a single container. Although these concepts are closely related, they solve different problems and are often confused.

Compression vs Archiving

Compression Archiving
Reduces file size. Combines multiple files.
May use CPU time. Improves organization.
Can be applied to a single file. Usually stores directories or collections.

Lossless vs Lossy Compression

Compression algorithms are generally divided into two categories.

LosslessLossy
Original data is recovered perfectly.Some information is discarded.
ZIP, 7z, Zstandard, GzipJPEG, MP3, AAC, H.264

Archives and backups should normally use lossless compression.

When Compression Does Not Help

Many modern media formats are already compressed.

  • JPEG images
  • MP3 and AAC audio
  • MP4 videos
  • Most game assets

Maximum compression is not always the best choice. Faster algorithms often provide a better balance between storage savings and processing time.

Archive Organization

Instead of storing thousands of unrelated files in one archive, create archives with a clear purpose. 

Archives/

├── Photos_2025.7z
├── Tax_Records.zip
├── Thesis_Backup.tar.zst
└── Drivers_Collection.zip

Key Takeaway

Compression reduces storage requirements, while archiving improves organization. Choosing the appropriate format depends on the type of data, desired compatibility, performance requirements and long-term preservation goals.
