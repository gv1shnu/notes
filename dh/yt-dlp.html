yt-dlp

Difficulty: Easy

The one tool to install first. yt-dlp is the maintained successor to youtube-dl, and despite the name it pulls video and audio from a thousand-plus sites — not just YouTube, but Vimeo, Twitch, SoundCloud, most news sites, and anywhere a <video> tag hides. It's a single command that has quietly replaced a dozen sketchy "download this video" websites.

Install

It's one binary or one pip install, on every OS — Windows, macOS, Linux, and Android (via Termux). 

# pick one
pip install yt-dlp
brew install yt-dlp        # macOS
winget install yt-dlp      # Windows
sudo apt install yt-dlp    # Debian/Ubuntu

Install ffmpeg alongside it — yt-dlp uses it to merge separate video/audio streams and to convert formats. Without ffmpeg you'll get whichever single stream already exists, which is usually not the good one.

The commands worth memorising

yt-dlp URL                          # grab the best quality it can
yt-dlp -F URL                       # list every available format first
yt-dlp -f "bv*+ba/b" URL            # best video + best audio, merged
yt-dlp -x --audio-format mp3 URL    # audio only, as mp3
yt-dlp -S "res:1080,ext:mp4" URL    # prefer 1080p mp4
yt-dlp -o "%(title)s.%(ext)s" URL   # name the file sensibly

Why it beats the website buttons

FeatureWhat it does
Playlists / channelshand it a playlist URL, it grabs the lot
--download-archiverecords what it fetched, so re-running skips duplicates — ideal for "subscribe to a channel"
--embed-subsbakes subtitles into the file
--sponsorblock-remove allstrips sponsor segments on the way in

The catch: sites change their guts constantly, and a download that worked yesterday can break today. The fix is always the same — update: yt-dlp -U (or re-pip). And the obvious one: download what you have the right to. yt-dlp is a faithful pipe; where the water goes is on you.
