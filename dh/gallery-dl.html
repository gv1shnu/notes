gallery-dl

Difficulty: Medium

yt-dlp is for things that move; gallery-dl is for things that don't. It's the image-and-gallery counterpart — point it at a profile, board, or album and it pulls every picture at full resolution, in order, with sane filenames. It speaks the layouts of Twitter/X, Instagram, Reddit, Pixiv, DeviantArt, imgur, Danbooru, and a long tail of others.

Install & basics

pip install gallery-dl

gallery-dl URL                # download the whole gallery
gallery-dl -g URL             # just print the media URLs, download nothing
gallery-dl --range "1-20" URL # only the first 20 items

The part that trips people: auth

Half of what's worth grabbing sits behind a login. gallery-dl doesn't want your password — it borrows your browser's cookies, so it's you asking, logged in as you. 

gallery-dl --cookies-from-browser firefox URL

For anything recurring, a config file at ~/.config/gallery-dl/config.json lets you set output folders, rename patterns, and per-site credentials once instead of retyping flags forever.

RealityWhat to do
Rate limits / soft bansslow down — --sleep between requests; don't hammer
Login-walled contentuse --cookies-from-browser; only what your account can already see
Site changed its layoutupdate: pip install -U gallery-dl

The catch: a public profile and a re-publishable one are different things. gallery-dl makes collecting trivial; whether you may keep or repost is a separate question the tool won't ask you.
