How do we create a database?

In MySQL, a database can be created directly using the following command: 


MySQL



CREATE DATABASE company;

PostgreSQL also allows a superuser to create a database directly: 


PostgreSQL (Superuser)



CREATE DATABASE company;

In larger systems, it is common to create a dedicated user who will own and manage the database instead of using the superuser account for day-to-day operations. 


PostgreSQL



CREATE USER rambo

WITH PASSWORD 'jumbo';






PostgreSQL


CREATE DATABASE company

WITH OWNER rambo;

The database owner is responsible for managing objects inside the database and can grant access to other users when required.

Note: Creating a dedicated owner is not mandatory. PostgreSQL databases can be created directly by a superuser. However, assigning ownership to a specific user is considered a good administrative practice, especially in production environments.

List the databases


MySQL


SHOW DATABASES;




PostgreSQL


\l

Important: PostgreSQL shorthand commands such as \l, \dt, \c, and others are available only within the PostgreSQL terminal client (psql). They are not part of standard SQL and therefore cannot be used in graphical tools such as DBeaver, pgAdmin, or other database clients.

How do we use the database?

We need to specify which database we are working with. 


MySQL


USE company;




PostgreSQL


\c company

Note: PostgreSQL has no USE statement.
