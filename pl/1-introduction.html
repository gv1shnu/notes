Introduction

Python is a high-level, interpreted language built around one stubborn priority: readability. You write something close to English, the interpreter runs it line by line, and a lot of the tedious bookkeeping (memory, types) is handled for you. The tax for that comfort is raw speed — but for learning, scripting, data, and glue code, it's the friendliest tool there is.

What makes it tick

Trait Means
Interpreted runs directly — no separate compile step; try things live in the REPL
Dynamically typed a variable's type is decided at runtime, not declared up front
Indentation = blocks whitespace isn't cosmetic — it is the structure (no braces)
Batteries included a huge standard library ships with it

The obligatory first line

print("Hello, world")

No main, no semicolons, no imports. The interpreter reads the file top to bottom and does what each line says. That immediacy — type a line, see a result — is the whole reason Python is where most people start.

These notes follow the natural build-up: the values you store (numbers, text), the containers you hold them in (lists and friends), the decisions and loops that make code do things, then functions and objects to organize it all.
