Loops

Doing something repeatedly. Python has two: for (walk over a collection) and while (repeat until a condition flips). The choice is simple — do you know what you're iterating over, or just when to stop?

for — iterate over items

for name in ["TenZ", "aspas", "yay"]:
    print(name)

for i in range(3):       # 0, 1, 2  (stop is excluded)
    print(i)

Python's for loops over the items directly — no manual index. Need the index too? Use enumerate; iterate two lists in lockstep with zip: 

for i, name in enumerate(players):     # (0, 'TenZ'), (1, 'aspas'), ...
    print(i, name)

for name, score in zip(names, scores):  # pair them up
    print(name, score)

while — repeat until done

n = 5
while n > 0:
    print(n)
    n -= 1            # must move toward the exit, or loop forever

Use while when the number of iterations isn't known up front. The cardinal sin: forgetting to change the condition variable — an infinite loop.

Steering a loop: break, continue, pass

Keyword Effect
break exit the loop entirely, right now
continue skip to the next iteration
pass do nothing — a placeholder where syntax needs a body 
for x in range(10):
    if x == 5:
        break          # stop at 5
    if x % 2 == 0:
        continue       # skip evens
    print(x)           # prints 1, 3

break is your early exit (e.g. stop once you've found what you searched for); continue filters out cases you don't care about; pass just fills a block you haven't written yet.
