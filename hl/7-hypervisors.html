Hypervisors

Difficulty: Intermediate

A hypervisor is software that creates and manages virtual machines. It controls how physical hardware is shared between multiple guest operating systems while keeping them isolated from one another. Every virtualization platform relies on a hypervisor.

Choosing a Hypervisor

  • Desktop learning → Type 2 platforms.
  • Dedicated server → Type 1 platforms.
  • Many virtual machines → Enterprise-style hypervisors.
  • Occasional testing → Simpler desktop solutions.

Key Takeaway

Hypervisors make virtualization possible by sharing hardware efficiently while keeping virtual machines isolated. Choosing the appropriate platform depends on whether the system is intended for experimentation, daily development or long-term server hosting.
