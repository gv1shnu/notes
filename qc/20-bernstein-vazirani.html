Bernstein–Vazirani

First real algorithm, and a clean win for quantum. The problem: there's a hidden n-bit string s, and an oracle that, given x, returns the bitwise dot product s · x (mod 2). Find s.

The query count

Approach Queries to recover s (n bits)
Guess randomly up to 2n
Classical, clever (probe one bit at a time) n
Bernstein–Vazirani (quantum) 1

Classically you must isolate each bit — query 1000... to read s's first bit, 0100... for the second, n queries total. Quantumly, one shot hands you the whole string.

The recipe

Step Action
1 H on all n input qubits (superposition over every x)
2 put the ancilla in |−⟩ (X then H) — arms phase kickback
3 apply the oracle (a CNOT to the ancilla from each qubit where s has a 1)
4 H on all n input qubits again, then measure → out comes s

Why it works: the oracle kicks back a phase of (−1)s·x onto every x in the superposition at once. The second layer of Hadamards interferes all those phases so that everything cancels except the single basis state equal to s. Skip the |−⟩ ancilla and you lose the kickback — the result decays to a useless 50/50 (a great thing to break on purpose to see why it's there). One oracle call, the entire hidden string.
