Controlled Gates: CNOT & Toffoli

Single-qubit gates rotate one qubit in isolation — they can't make two qubits care about each other. Entanglement needs a gate where one qubit's value decides what happens to another. That's a controlled gate.

CNOT — controlled NOT

Two qubits: a control and a target. The rule is one line:

if control = |1⟩, flip the target; otherwise do nothing.

Input Output
|00⟩ |00⟩
|01⟩ |01⟩
|10⟩ |11⟩
|11⟩ |10⟩

Read the target as control XOR target — the same exclusive-or from classical logic, now reversible.

The greatest hit: the Bell state

Hadamard then CNOT is the two-line recipe that produces entanglement, and you will type it a hundred times:

Step State
Start |00⟩
H on q0 (1/√2)(|00⟩ + |10⟩) — q0 in superposition
CNOT (q0→q1) (1/√2)(|00⟩ + |11⟩) — entangled

That's the Bell state from the end of Course 1, now built rather than just admired. The superposition feeds CNOT both possibilities at once; CNOT ties them together so the only survivors are 00 and 11. Measure one qubit, instantly know the other.

Toffoli — CNOT with a plus-one

The Toffoli gate (CCX) is just CNOT with two controls: flip the target if and only if both controls are |1⟩. Once you know CNOT, you know this one for free — it's a reversible AND, and it's enough to rebuild any classical circuit on a quantum computer.

The intuition that pays off: every controlled gate is "check the control(s), conditionally act on the target." Meet a strange new gate in the Qiskit docs and that sentence (plus a matrix multiply) decodes it. Time to actually run one.
