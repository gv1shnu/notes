17. threading.

Start a thread by handing it a function plus its arguments. Then guard shared data with locks.

Primitive Use
start_new_thread(fn, args) run fn in a new thread
lock.acquire() / release() one thread in the critical section at a time
Semaphore let a fixed number of threads in at once
Event signal between threads
Barrier all threads wait here until everyone arrives

A barrier is the clean way to make a value coherent across threads before anyone moves on.

In code 
Python

import threading

lock = threading.Lock()
count = 0

def worker():
    global count
    with lock:              # acquire + release
        count += 1

threads = [threading.Thread(target=worker) for _ in range(4)]
for t in threads: t.start()
for t in threads: t.join()
print(count)                  # 4, safely
