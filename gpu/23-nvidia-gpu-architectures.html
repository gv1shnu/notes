23. Nvidia architectures.

Seven generations of CUDA-capable hardware, each roughly two years. The pattern: more cores, less power — so performance per watt keeps doubling.

Generation Years Notable
Tesla 2008 first CUDA architecture
Fermi 2010 4× cores vs Tesla
Kepler 2012 more programmable, lower power
Maxwell 2014 ~2× cores, better bandwidth
Pascal 2016 unified memory, NVLink (fast host↔device)
Turing 2018 ray tracing
Ampere 2020 tensor cores; ~4× performance per watt
