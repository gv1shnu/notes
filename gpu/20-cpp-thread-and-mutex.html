Parallel syntax — C++

The C++ standard library (std::) has four concurrency tools: thread, mutex, atomic, future. First two here.

20. thread & mutex.

std::thread Effect
std::thread t(fn) start a thread running fn
t.join() caller waits until the thread finishes
t.detach() thread runs free, no longer tied to the caller
std::mutex Effect
lock() block others, enter the critical section
unlock() release when done
try_lock() attempt the lock; if busy, carry on instead of waiting

try_lock + sleep can limit thrashing over a hot section — but sleep too long and the thread sits underused.

In code 
C++

#include <thread>
#include <mutex>

std::mutex m;
int count = 0;

void worker() {
    m.lock();
    ++count;
    m.unlock();
}

int main() {
    std::thread t1(worker), t2(worker);
    t1.join();
    t2.join();
}
