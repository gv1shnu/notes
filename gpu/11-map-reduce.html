Concurrency patterns

11. Map-reduce.

A subset of divide and conquer. Break the data into single points, apply the same operation to each (map), then fold the results into one value (reduce).

Example: is X in the set? Each mapper returns 1 on a match, else 0. The reducer sums them.
Element Mapper (X = 7)
3 0
7 1
5 0
reduce (sum) 1 → true

Scales well at any data size, as long as the system is well built and communication is not slow. Jobs can be chained together.
