Parallel syntax — Python 3

18. asyncio.

For code that waits on something slow (a network reply) without freezing the whole program. Mark a function async; the library guarantees it eventually completes.

Keyword / call Effect
async def marks a coroutine that may run asynchronously
asyncio.run(coro) run the coroutine
await wait here for a result before continuing
create_task(coro) run it and return a task object you can read the result from
gather(...) run several tasks in parallel
sleep(t) pause a coroutine for t seconds

Before this, Python 2 web code either blocked or was monkey-patched — not pretty.

In code 
Python

import asyncio

async def fetch(n):
    await asyncio.sleep(1)       # pretend network wait
    return n * 2

async def main():
    results = await asyncio.gather(fetch(1), fetch(2), fetch(3))
    print(results)              # [2, 4, 6] in ~1s, not 3s

asyncio.run(main())
