25. Compilation with nvcc.

nvcc compiles GPU and host code into one executable, passing through intermediate forms.

Intermediate What it is
PTX virtual assembly — portable across hardware
cubin real binary for a specific architecture
fatbin a bundle of the above
Option Use
-h help / usage
-o name the output
--lib compile into a library for shared projects
--run compile, link, and execute in one step
-arch / -code / -gencode target a hardware architecture (else you get deprecation warnings)

Virtual architecture = which features (compute version); real architecture = which physical hardware.

In code 
Bash

nvcc -o vector_add vector_add.cu     # compile
./vector_add                         # run

nvcc --run vector_add.cu             # compile + run in one step
nvcc -arch=sm_62 -o vector_add vector_add.cu   # target a specific GPU
