Flynn's taxonomy pointed at SIMD hardware. That hardware is the GPU. First, which kind you have.

22. Integrated vs dedicated.

Integrated Dedicated
Where on the CPU chip, shares system memory separate card, own memory
Power / heat low — good for battery high
Compute weak strong
CUDA? no (use OpenCL / OpenACC) yes, on Nvidia

Frameworks for talking to a GPU:

Framework Scope
CUDA Nvidia only; most widely adopted, deepest access
OpenCL many hardware platforms
OpenACC macros around code to offload to accelerators
Metal Apple's own (Macs ship AMD, not Nvidia)

Heavy users: crypto mining, NLP, computer vision — neural nets map well to "many cores, each doing a little".
