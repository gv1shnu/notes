9. reduce.

Fold every element into a single value using a two-argument function. An action.

In code 
Python

from operator import add

nums = sc.parallelize([1, 2, 3, 4, 5])

nums.reduce(add)                     # 15  (1+2+3+4+5)
nums.reduce(lambda x, y: x * y)     # 120 (1*2*3*4*5)
