11. cache & persist.

By default an RDD is recomputed from its lineage every time you run an action on it. If you reuse it, that's wasteful — cache / persist keep it in memory for next time.

In code 
Python

words.cache()           # keep in memory
words.persist()         # same, but you can pick the storage level
words.is_cached         # True
Call Difference
cache() memory only
persist(level) choose where: memory, disk, or both

Cache an RDD you'll hit more than once; skip it for a one-shot.
