12. DataFrame joins.

We joined RDDs by hand earlier; on a DataFrame it's one call. df.join takes three things: the right DataFrame, the column(s) to join on, and the join type.

In code 
Python

tableA.join(tableB,
            tableA.id == tableB.id,
            "inner").show()
Type Keeps
inner only rows that match on both sides
outer all rows; nulls where there's no match
left all left rows (+ matches)
right all right rows (+ matches)
left-semi left rows that have a match (left columns only)

When both tables share a column name, alias them so you can tell the columns apart: 

Python

ta = tableA.alias("ta")
tb = tableB.alias("tb")
# now refer to ta.name vs tb.name
