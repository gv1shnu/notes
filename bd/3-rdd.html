3. RDD.

Resilient Distributed Dataset — Spark's core data structure. An immutable collection of objects, split into partitions that live on different nodes of the cluster.

Property Meaning
Resilient remembers how it was built (lineage); can rebuild a lost partition after a failure
Distributed partitioned across many nodes, computed in parallel
Immutable never changed in place — every operation makes a new RDD

Because each RDD knows the one it came from, Spark can recreate it at any point if something breaks. That lineage is what "resilient" buys you.
