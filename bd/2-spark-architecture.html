2. Architecture.

Spark Core is the backbone: in-memory computation, parallel/distributed execution, I/O, fault tolerance, memory management. Everything else is built on top of it.

Component For
Spark SQL structured data — a distributed SQL query engine
Spark Streaming real-time data, in fault-tolerant micro-batches
MLlib machine learning at scale
GraphX / GraphFrames graph analytics
Spark SQL Streaming MLlib GraphX
Spark Core

At runtime the driver holds your program and the SparkContext; it splits the job into tasks and ships them to executors running on worker nodes, each crunching its slice in parallel and caching results in memory. Spark Core is what makes that distribution and fault tolerance work.
