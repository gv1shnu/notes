8. foreach.

The simplest action: run a function on every element for its side effect (print, write out, etc.). It returns nothing.

In code 
Python

words = sc.parallelize(["scala", "java", "spark"])

def f(x):
    print(x)

words.foreach(f)        # prints each element

Use map when you want a new RDD back; use foreach when you only want the side effect.
