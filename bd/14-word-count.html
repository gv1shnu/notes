14. Word count.

The "hello world" of big data — and it stitches together everything so far: a transformation chain ending in one action.

In code 
Python

lines = sc.textFile("input.txt")

counts = (lines
          .flatMap(lambda line: line.split(" "))   # line → words
          .map(lambda w: (w, 1))                    # word → (word, 1)
          .reduceByKey(lambda a, b: a + b))        # sum per word

counts.collect()        # [('spark', 2), ('hadoop', 2), ...]

For a quick tally without the chain, countByValue does the same for small data: 

Python

words.countByValue()    # {'Spark': 2, 'Hadoop': 2, 'Pig': 1}

Watch out: PySpark is case-sensitive — "Spark" and "spark" count as different words.
