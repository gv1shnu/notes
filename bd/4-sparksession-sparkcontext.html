4. SparkSession & SparkContext.

Two entry points. SparkSession is the modern one for DataFrames and SQL; SparkContext sits underneath it for raw RDDs.

In code 
Python

from pyspark.sql import SparkSession

spark = (SparkSession.builder
         .appName("my-app")
         .getOrCreate())          # reuse if it exists, else create

sc = spark.sparkContext          # the SparkContext, for RDDs

getOrCreate() means you never accidentally spin up two sessions.
