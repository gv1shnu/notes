13. MySQL connectivity.

We've built DataFrames from collections and files; a database is just another source. Read a table straight into a DataFrame over JDBC. One setup step: drop the MySQL JDBC .jar into Spark's jars folder so it's on the path.

In code 
Python

df = (spark.read.format("jdbc")
      .option("url", "jdbc:mysql://localhost:3306/demo1")
      .option("driver", "com.mysql.jdbc.Driver")
      .option("dbtable", "demo_table")
      .option("user", "root")
      .option("password", "****")
      .load())

df.show()           # the table, in PySpark
df.printSchema()    # columns, types, nullable
Option Is
url jdbc:mysql://host:3306/database
driver the JDBC driver class
dbtable which table to read
