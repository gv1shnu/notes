5. Creating data.

Three ways to get data into Spark: parallelize a Python collection, build a DataFrame from rows, or read a file.

In code 
Python

# 1. from a collection → RDD
rdd = sc.parallelize([1, 2, 3, 4, 5])

# 2. from rows + column names → DataFrame
df = spark.createDataFrame(
    [(1, "Henry", 50000), (2, "Sam", 60000)],
    ["id", "name", "salary"])

# 3. from a CSV file → DataFrame
df = (spark.read.format("csv")
      .option("header", True)
      .option("inferSchema", True)
      .load("data.csv"))
Option Effect
header=True first row is column names
inferSchema=True guess column types instead of all-string
